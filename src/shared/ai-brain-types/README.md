This folder contains the auto-generated types coming from the backend.

1. Go to backend folder:

   N:\\nima-ai-marketing

2. Run the generator:

   python scripts/generate-types.py

3. Copy the generated file:

   N:\\nima-ai-marketing\\web\\types-generated.js

4. Paste it here:

   C:\\Users\\USER\\Documents\\GitHub\\www.nimasaraeian.com\\src\\shared\\ai-brain-types\\types-generated.js

5. DONE — frontend now knows the updated API structure.

NEVER edit types-generated.js manually.
